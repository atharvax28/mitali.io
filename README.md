# mitali.io

A fast, responsive product/portfolio site built on a Radix UI + Tailwind design system (shadcn/ui), bundled with Vite.

🔗 **Live:** https://mitali-io.vercel.app

---

## Highlights

- **Vite + React + TypeScript** for an instant dev loop and a lean production bundle
- **shadcn/ui** — accessible Radix primitives (dialog, dropdown, accordion, hover-card, and more) styled with Tailwind
- **Forms** with `react-hook-form` + resolver validation
- Component-driven, fully responsive layout

## Tech stack

`React` · `TypeScript` · `Vite` · `Tailwind CSS` · `Radix UI / shadcn/ui` · `react-hook-form`

## Getting started

```bash
npm install
npm run dev      # start the dev server
npm run build    # production build
npm run preview  # preview the build
```
